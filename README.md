## Run Locally

### Option 1: Just open the file
1. Clone the repo:
   ```bash
   git clone https://github.com/Induranga-kawishwara/portfolio.git
   ```
2. Open `index.html` in your browser.

### Option 2: Use VS Code Live Server (recommended)
1. Install **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**

## Deploy (GitHub Pages)
This site is hosted with **GitHub Pages**.
1. Go to **Repo Settings → Pages**
2. Under **Build and deployment**, select:
   - **Branch:** `main`
   - **Folder:** `/ (root)`
3. Save — your site will be published.

## Customization
To edit the content:
- Update text/sections inside `index.html`
- Replace images inside `assets/`
- Update the **Projects** and **Skills** content inside the HTML/JS used in the site
- If you fork this repo, remember to change:
  - Name
  - Links 
  - Email/contact details
  - CV file
